Google Chrome Puppet Module
===========================

Overview
--------

Install Google Chrome from the official Google repository.


Usage
-----

Example:

    class { 'googlechrome': }


Supported Platforms
-------------------

The module has been tested on the following operating systems. Testing and patches for other platforms are welcomed.

* Fedora 17
